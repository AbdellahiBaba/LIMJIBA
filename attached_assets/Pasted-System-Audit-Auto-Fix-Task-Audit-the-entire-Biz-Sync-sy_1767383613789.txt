System Audit + Auto‑Fix
Task:  
Audit the entire Biz Sync system (backend, frontend, database logic, API routes, and utility functions).
Verify whether the following business rules are implemented correctly.
If any part is missing, incorrect, or incomplete, fix it automatically.

1. Fabrication Invoice Logic (Must Be Corrected if Wrong)
Fabrication invoices represent manufacturing cost, not revenue.
The system must follow these rules:

Fabrication invoices must NOT be added to Invoice Revenue.

Fabrication cost must be treated as COGS (Cost of Goods Sold).

After fabrication, the produced items must be added to inventory with:

Cost Price = total fabrication cost ÷ quantity produced

Sell Price = predefined selling price

Fabrication cost must flow into:

Inventory valuation

COGS

Profit calculations (as an expense, not revenue)

If any of these rules are not implemented, correct them.

2. Salary Payments & Fabrication Costs → Neon Database Only
Check whether salary payments and fabrication expenses are being saved to the Neon production database.

If any part of the system still writes to a dev/local database:

Remove all dev DB references

Update environment variables

Ensure all CRUD operations write only to Neon

3. Full Migration to Neon Database
Verify that all operations (sales, purchases, salaries, fabrication, inventory updates, reseller operations, expenses) are stored in Neon.

If any operation still uses a dev DB or fallback DB, fix it.

4. Profit Calculation Must Follow Standard Accounting
Ensure the system uses the correct formulas:

Gross Profit
Revenue – COGS

Operating Profit
Gross Profit – Operating Expenses

Net Profit
Operating Profit – Taxes (if applicable)

COGS must include fabrication cost.

If the current implementation differs, correct it.

5. Validation & Testing
After fixing, the agent must:

Test revenue calculations

Test fabrication → inventory flow

Test salary expense flow

Test Neon DB writes

Confirm no dev DB writes remain

Confirm fabrication invoices never appear in revenue

Goal:
Ensure the system is fully correct, financially accurate, and permanently migrated to Neon as the single source of truth.
