Environment File (/.env) is accessible on your server

Exposed Environment File can reveal sensitive configuration, credentials, or allow unauthorized access to admin tools.

Block Path
A05:2021 Security Misconfiguration
CWE-200
CVSS: 9.5
Remediation

Remove .env from web root. Add to .gitignore. Use environment variables via hosting platform.

Config Fix

# Nginx
location ~ /\.env { deny all; return 404; }
# Apache
<Files .env>
  Require all denied
</Files>
